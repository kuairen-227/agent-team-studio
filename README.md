# Agent Team Studio

AI駆動開発の実践プロジェクト。企画から運用保守まで一気通貫でAIと協働して回すことを目指します。

## プロジェクトの目的

- **学習目的**: 最新のベストプラクティスに沿った技術スタック・アーキテクチャを、自ら理解しながら構築する
- **AI駆動開発の実践**: 企画→要件定義→設計→開発→試験→改善→運用保守のサイクルを、マネジメント業務含めてAIと協働で回す
- **意思決定の記録**: 設計判断の背景と理由をADRとして残し、知見を蓄積する

## 状況設定

本プロジェクトでは、リアリティある開発プロセスを体験するため以下の仮想状況を設定しています（[ADR-0002](docs/adr/0002-project-scenario.md)）：

- **シナリオ**: 社内新規事業として新プロダクトを立ち上げる
- **チーム**: PM 1人 + エンジニア 1人（エンジニアが AI と協働して開発）
- **制約**: 標準的な社内制約（クラウド承認制、セキュリティレビューあり、ただし柔軟性あり）
