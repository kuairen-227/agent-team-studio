# 用語集（MVP）

doc / code / UI ラベルで使用する用語を統一し、命名の揺れと参照ズレを防ぐための辞書。英語名はコード上の識別子、日本語は UI・ドキュメント表記として用いる。

本用語集の範囲は MVP（[ADR-0005](../adr/0005-mvp-scope.md)）で頻出するものに限定する。詳細な技術用語は `docs/design/` 配下の各 doc で個別に定義する方針。

## 使い方

- **コード**: 英語名（PascalCase / camelCase）を型・関数・テーブル名に使用
- **ドキュメント**: 日本語を本文に使用。初出時のみ英語を併記（例: 「実行（Execution）」）
- **UI**: 日本語を表示。ボタン文言・ステータスラベルは本ドキュメントの UI 用語節に従う
- 揺れを発見した場合は本ドキュメントに集約し、各参照を統一する

### 更新ポリシー

- 2 箇所以上の doc / code / UI で参照される語は本ファイルに集約する
- ドメイン語（product 観点の概念）は本ファイルへ追記。技術用語は各 design doc 内で定義し、本ファイルには最小限のみ置く
- 英語名が未確定の用語は `—` を入れ、design doc で確定次第更新する

## テーブル形式

各セクションのテーブルは以下の 4 列で統一する。該当値がない列は `—` を入れる。

- **英語名**: コード上の識別子（PascalCase / camelCase / 単語）。未確定は `—`
- **日本語**: UI・ドキュメント本文の表記
- **定義**: 意味・使い方・表記ルール（不使用語を含む）
- **関連**: 一次ソース ADR / doc / US

## UI 用語

画面・ボタン・ステータスラベルの表記統一。[screen-flow.md](./screen-flow.md) と連動。

### 画面名

| 英語名 | 日本語 | 定義 | 関連 |
| --- | --- | --- | --- |
| — | テンプレート一覧 | テンプレートを選択する画面の名称。「テンプレート選択」等と表記しない（選択行為と独立した画面名） | [screen-flow.md](./screen-flow.md) |
| — | 入力フォーム | パラメータ入力画面の名称。「パラメータ入力」等と表記しない | [screen-flow.md](./screen-flow.md) |
| — | 進捗 | 実行中のエージェント状態を表示する画面の名称。「実行中画面」等と表記しない | [screen-flow.md](./screen-flow.md) |
| — | 結果 | 統合レポート表示画面の名称。「レポート」「完了画面」等と表記しない | [screen-flow.md](./screen-flow.md) |
| — | 履歴一覧 | 過去実行一覧画面の名称。「実行履歴」単独では画面を指さず、履歴データ（Execution History）そのものを指す | [screen-flow.md](./screen-flow.md) |

コード上の画面識別子（ルーティングキー等）は design doc で確定する。
