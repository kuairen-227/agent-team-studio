# 用語集（MVP）

doc / code / UI ラベルで使用する用語を統一し、命名の揺れと参照ズレを防ぐための辞書。英語名はコード上の識別子、日本語は UI・ドキュメント表記として用いる。

本用語集の範囲は MVP（[ADR-0005](../adr/0005-mvp-scope.md)）で頻出するものに限定する。詳細な技術用語は `docs/design/` 配下の各 doc で個別に定義する方針。

## 使い方

- **コード**: 英語名（PascalCase / camelCase）を型・関数・テーブル名に使用
- **ドキュメント**: 日本語を本文に使用。初出時のみ英語を併記（例: 「実行（Execution）」）
- **UI**: 日本語を表示。ボタン文言・ステータスラベルは本ドキュメントの UI 用語節に従う
- 揺れを発見した場合は本ドキュメントに集約し、各参照を統一する

## プロダクトドメイン用語

MVP のコア概念。ADR-0005 の MVP 機能一覧と [user-stories.md](./user-stories.md) に準拠。

| 英語名 | 日本語 | 定義 | 関連 |
| --- | --- | --- | --- |

| Execution | 実行 | ユーザーがテンプレートを選択しパラメータを入力して開始した 1 回のエージェントチーム実行。開始→完了の一方向（中断・再開は Non-goal） | [US-2](./user-stories.md#us-2-調査パラメータを入力して実行する), [US-3](./user-stories.md#us-3-エージェントの進捗をリアルタイムで見る) |

| Reference | 参考情報 | ユーザーが入力フォームで任意添付するテキスト。LLM に渡る調査コンテキストとして利用される。URL を貼り付けた場合も Web 取得は行わず、入力文字列をそのまま LLM に渡す | [US-2](./user-stories.md#us-2-調査パラメータを入力して実行する) |

| Matrix | マトリクス | 結果の固定フォーマット。行＝観点、列＝競合企業。カスタマイズは v2 以降 | [US-4](./user-stories.md#us-4-統合結果を閲覧しエクスポートする) |
| Execution History | 実行履歴 | 過去の実行の記録。テンプレート名・実行日時・ステータス・結果を保持する | [US-5](./user-stories.md#us-5-過去の実行履歴を振り返る) |

## UI 用語

画面・ボタン・ステータスラベルの表記統一。[screen-flow.md](./screen-flow.md) と連動。

### 画面名

| 表記 | 備考 |
| --- | --- |
| テンプレート一覧 | 「テンプレート選択」等と表記しない。選択行為とは独立した画面名 |
| 入力フォーム | 「パラメータ入力」等と表記しない |
| 進捗 | 「実行中画面」等と表記しない |
| 結果 | 「レポート」「完了画面」等と表記しない |
| 履歴一覧 | 「実行履歴」単独では画面を指さず、履歴データそのものを指す |

### アクション語

| 表記 | 不使用 | 備考 |
| --- | --- | --- |
| 実行 | 開始、スタート | ボタン文言・動詞表記を「実行」で統一（例: 「実行する」「実行を開始」） |
| エクスポート | 書き出し、出力 | Markdown 持ち出し全般を指す |
| コピー | — | クリップボードへの Markdown コピー |
| ダウンロード | セーブ、保存 | `.md` ファイル保存 |

### ステータスラベル

エージェントおよび実行のステータス表示（[US-3](./user-stories.md#us-3-エージェントの進捗をリアルタイムで見る) 受け入れ基準に準拠）。

| 表記 | 英語名 | 意味 |
| --- | --- | --- |
| 待機 | pending | 実行順序待ちで未着手 |
| 実行中 | running | LLM 呼び出し中もしくはストリーミング出力中 |
| 完了 | completed | 正常終了 |
| 失敗 | failed | エラーにより終了（原因メッセージを併記） |

### その他

| 表記 | 備考 |
| --- | --- |
| 空状態 | 履歴 0 件時等に表示するプレースホルダ文言の総称 |
| ドッグフーディング | チーム自身がユーザーとしてプロダクトを使用して検証する活動（[ADR-0004](../adr/0004-target-users.md)） |
